SUBOUTINE RVSUB - MANUAL PAGE

 DESCRIPTION
 Subtracts two double precision vectors                             
                                                                    
 This function subtracts two double precision vectors passed as     
 arguments and stores the result in another vector U=V-W.           
                                                                    

 ARGUMENT LIST
 Type               Name      Description

 double precision   U      <  Double precision vector with the result 
                              V-W.                                    

 double precision   V      >  Double precision vector.                

 double precision   W      >  Double precision vector.
