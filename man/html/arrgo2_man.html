SUBOUTINE ARRGO2 - MANUAL PAGE

 DESCRIPTION
 Arrange a fourth order tensor in matrix form with G matrix ordering
                                                                    
 This routine re-arranges a given fourth order tensor, stored as a  
 4-index array, in matrix form (2-index array) using G matrix       
 component ordering.                                                
 Implemented only for 2-D tensors.                                  
                                                                    

 ARGUMENT LIST
 Type               Name      Description

 double precision   A4TH   >  Fourth order tensor stored as a 4-index 
                              array.                                  

 double precision   AMATX  <  2-index array containing the components 
                              of the given 4th order tensor stored    
                              using G matrix ordering.
