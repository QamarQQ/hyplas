SUBOUTINE RDDMEL - MANUAL PAGE

 DESCRIPTION
 Read data for damaged elastic/crack closure model.                 
                                                                    
 This routine reads from the data file and echos to the results file
 the material parameters necessary for the isotropically damaged    
 isotropic elastic model accounting for partial microcrack/void     
 closure effects (quasi-unilateral conditions) under compressive    
 stresses.                                                          
                                                                    

 ARGUMENT LIST
 Type               Name      Description

 integer            NTYPE  >  Stress state type flag.                 

 double precision   RPROPS <  Array of real material properties.      

 logical            UNSYM  <  Unsymmetric tangent stiffness flag.
