SUBOUTINE EXT3 - MANUAL PAGE

 DESCRIPTION
 Gauss point-node extrapolation matrix for element type TRI3        
                                                                    
 This routine sets the coefficients matrix for extrapolation of     
 fields from Gauss point values to nodal values for element type    
 TRI3: Standard isoparametric 3-noded linear triangle.              
                                                                    

 ARGUMENT LIST
 Type               Name      Description

 double precision   EXMATX <  Extrapolation matrix.
