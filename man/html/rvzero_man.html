SUBOUTINE RVZERO - MANUAL PAGE

 DESCRIPTION
 Zero a double precision array                                      
                                                                    
 This routine sets to zero the N components of the double precision 
 array argument V.                                                  
                                                                    

 ARGUMENT LIST
 Type               Name      Description

 double precision   V      <  Zeroed double precision array.          

 integer            N      >  Dimension of V.
