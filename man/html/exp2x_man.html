DOUBLE PRECISION FUNCTION EXP2X - MANUAL PAGE

 DESCRIPTION
 f(x)=exp(2x)                                                       
                                                                    
 This function relates the eigenvalues of the Cauchy-Green strain   
 tensors to the principal logarithmic stretches.                    
                                                                    

 ARGUMENT LIST
 Type               Name      Description

 double precision   X      >  Point at which the function will be     
                              evaluated.
