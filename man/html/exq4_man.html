SUBOUTINE EXQ4 - MANUAL PAGE

 DESCRIPTION
 Sets Gauss point-node extrapolation matrix for element type QUAD4  
                                                                    
 This routine sets the coefficients matrix for extrapolation of     
 fields from Gauss point values to nodal values for element type    
 QUAD4: Standard isoparametric 4-noded bi-linear quadrilateral.     
                                                                    

 ARGUMENT LIST
 Type               Name      Description

 integer            NGAUSP >  Number of Gauss points.                 

 double precision   EXMATX <  Extrapolation matrix.
