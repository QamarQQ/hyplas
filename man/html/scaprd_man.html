DOUBLE PRECISION FUNCTION SCAPRD - MANUAL PAGE

 DESCRIPTION
 Scalar product of double precision vectors                         
                                                                    
 This function returns the scalar product between its two double    
 precision vector arguments U and V.                                
                                                                    

 ARGUMENT LIST
 Type               Name      Description

 double precision   U      >  Array of components of a double         
                              precision vector.                       

 double precision   V      >  Array of components of a double         
                              precision vector.                       

 integer            N      >  Dimension of U and V.
