SUBOUTINE FOPEN - MANUAL PAGE

 DESCRIPTION
 Opens/sets the names of the input data, results and re-start files 
                                                                    
 This routine reads the input data file name from the standard      
 input, sets the names and opens the corresponding data and results 
 files, and then sets the re-start file name.                       
                                                                    

 ARGUMENT LIST
 Type               Name      Description

 character           DATFIL <  Character string containing the input  
                               data file name.                        

 character           RESFIL <  Character string containing the results
                               output file name.                      

 character           RSTOUT <  Character string containing the        
                               re-start output file name.
