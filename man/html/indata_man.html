SUBOUTINE INDATA - MANUAL PAGE

 DESCRIPTION
 Reads most of the input data from a data file                      
                                                                    

 ARGUMENT LIST
 Type               Name      Description

 integer            MXFRON <  Maximum front size encountered in the   
                              specified mesh.                         

 logical            UNSYM  <  Global unsymmetric tangent stiffness    
                              flag. Set to .TRUE. if any              
                              element and/or material produces an     
                              unsymmetric tangent stiffness requiring 
                              the unsymmetric solver during           
                              equilibrium iterations.
