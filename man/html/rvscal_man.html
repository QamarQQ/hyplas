SUBOUTINE RVSCAL - MANUAL PAGE

 DESCRIPTION
 Multiplies a double precision vector by a scalar.                  
                                                                    

 ARGUMENT LIST
 Type               Name      Description

 double precision   V      <> Double precision vector.                

 integer            N      >  Dimension of V.                         

 double precision   SCAL   >  Scalar by which V will be multiplied.
