SUBOUTINE PODEC2 - MANUAL PAGE

 DESCRIPTION
 Polar decomposition of 2-D tensors                                 
                                                                    
 This routine performs the right polar decomposition of 2-D         
 tensors:   F = R U, where R is a rotation (orthogonal tensor)      
 and U is a symmetric tensor.                                       
                                                                    

 ARGUMENT LIST
 Type               Name      Description

 double precision   F      >  2-D tensor to be decomposed.            
                              Dimension 2x2.                          

 double precision   R      <  Rotation matrix resulting from the polar
                              decomposition. Dimension 2x2.           

 double precision   U      <  Right symmetric tensor resulting from   
                              the polar decomposition. Dimension 2x2.
