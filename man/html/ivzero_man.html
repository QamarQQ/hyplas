SUBOUTINE IVZERO - MANUAL PAGE

 DESCRIPTION
 Zero an integer array                                              
                                                                    
 This routine initialises to zero the N components of the integer   
 array argument IV.                                                 
                                                                    

 ARGUMENT LIST
 Type               Name      Description

 integer            IV     <  Zeroed integer array.                   

 integer            N      >  Dimension of IV.
