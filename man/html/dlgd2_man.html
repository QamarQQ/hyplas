DOUBLE PRECISION FUNCTION DLGD2 - MANUAL PAGE

 DESCRIPTION
 dlg2(x)=log(x)/2                                                   
                                                                    
 This function relates the principal logarithmic stretches and the  
 eigenvalues of the Cauchy-Green strain tensor.                     
                                                                    

 ARGUMENT LIST
 Type               Name      Description

 double precision   X      >  Point at which the function will be     
                              evaluated.
