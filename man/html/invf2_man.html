SUBOUTINE INVF2 - MANUAL PAGE

 DESCRIPTION
 Inverts the deformation gradient for 2-D problems                  
                                                                    
 This routine inverts deformation gradient tensors for plane strain,
 plane stress and axisymmetric problems.                            
                                                                    

 ARGUMENT LIST
 Type               Name      Description

 double precision   F      >  Deformation gradient.                   

 double precision   FINV   <  Inverse of the deformation gradient.    

 integer            NTYPE  >  Stress state type flag.
