SUBOUTINE ORDMEL - MANUAL PAGE

 DESCRIPTION
 Output results for the damaged elastic/crack closure model.        
                                                                    
 This routine writes to the results file some state variables for   
 the isotropically damaged isotropic elastic model accounting for   
 partial microcrack/void closure effects (quasi-unilateral          
 conditions) under compressive stresses.                            
                                                                    

 ARGUMENT LIST
 Type               Name      Description

 integer            NOUTF  >  Results file unit identifier.           

 integer            NTYPE  >  Stress state type flag.                 

 double precision   STRES  >  Array of stress tensor components.
