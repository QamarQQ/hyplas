DOUBLE PRECISION FUNCTION DDLGD2 - MANUAL PAGE

 DESCRIPTION
 ddlg2(x)=1/(2x)                                                    
                                                                    
 This is the derivative of the function defined in DLGD2,           
 that relates the principal logarithmic stretches and the           
 eigenvalues of the Cauchy-Green strain tensor.                     
                                                                    

 ARGUMENT LIST
 Type               Name      Description

 double precision   X      >  Point at which the function will be     
                              evaluated.
