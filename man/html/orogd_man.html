SUBOUTINE OROGD - MANUAL PAGE

 DESCRIPTION
 Output results for the Ogden hyperelastic material model           
                                                                    
 This routine writes to the results file state variables for the    
 Ogden hyperelastic material model. The state update procedure      
 for this material model is carried out in subroutine SUOGD.        
                                                                    

 ARGUMENT LIST
 Type               Name      Description

 integer            NOUTF  >  Results file unit identifier.           

 integer            NTYPE  >  Stress state type flag.                 

 double precision   STRES  >  Array of stress tensor components.
